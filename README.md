# Caesar Cipher Project 🔐

A simple Python project that implements the Caesar Cipher algorithm for encrypting and decrypting text. This project is designed for beginners to understand the basics of cryptography using Python.

---

## 📖 Description

The Caesar Cipher is one of the oldest and simplest encryption techniques. It works by shifting each letter in the plaintext by a fixed number of positions in the alphabet.

This project allows users to:
- Encrypt plain text
- Decrypt encrypted text
- Enter a custom shift key
- Learn the fundamentals of classical cryptography

---

## ✨ Features

- 🔒 Encrypt text
- 🔓 Decrypt text
- Custom shift key
- Simple command-line interface
- Beginner-friendly Python code
- Supports both uppercase and lowercase letters
